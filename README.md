# API FastAPI (Python)
API de para CRUD de produtos utilizando FastAPI e MySQL 

# Como Rodar

Clonar repositório
```cmd
git clone https://github.com/JoaoEnrique/conversor-fastapi.git
``` 

Instalar dependências
```cmd
pip install requirements.txt
```

Rodar projeto
```cmd
uvicorn main:app --reload
```
